# Personal Portfolio - Vue 3

A modern, elegant personal portfolio website built with Vue 3, TypeScript, and Tailwind CSS v4. Features dynamic theming, smooth animations, and a minimalist design aesthetic.

## Features

- **Dynamic Theming** - Switch between light, dark, and system themes
- **Accent Color Picker** - Customize the accent color in real-time
- **Smooth Animations** - Powered by Motion One for Vue
- **Responsive Design** - Optimized for all screen sizes
- **Keyboard Controls** - Navigate with keyboard shortcuts
- **Audio Feedback** - Optional sound effects for interactions
- **Visit Tracking** - Built-in analytics with Pinia state management

## Tech Stack

- **Frontend**: Vue 3.5.24 (Composition API), TypeScript 5.9.3
- **State Management**: Pinia 3.0.4
- **Styling**: Tailwind CSS v4
- **Build Tool**: Vite 7.2.4
- **Icons**: Lucide Vue Next
- **Utilities**: VueUse 14.1.0
- **Animations**: Motion One Vue 10.16.4

## Sections

- **Hero** - Name, title, and social links
- **Core Focus** - Systems Architecture and Product Strategy
- **Selected Work** - Project showcase
- **Chronicle** - Professional timeline

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customization

To personalize this portfolio for your own use:

1. Update the `experience` array in `App.vue` with your work history
2. Replace the "Selected Work" section with your actual projects
3. Update social media links in the hero section
4. Customize accent colors and branding in the store
5. Add your actual content to the services section
